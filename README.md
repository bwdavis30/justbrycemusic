The JustBryceMusic website is written and developed by me, single-developer Bryce Davis, as an entry-level percussion learning hub, centered around my teaching philosophies to those who desire to strengthen their skills on drumming. This project is ultimately created to brand myself as a musician and use as a personal portfolio.
On the website, the only page that requires any installation or downloading is on the Sheet Music page, which is luckily handled by the browser. However, no setup is required for basic functionality of the website.
To properly use the website, load the site into any modern browser, and use the navigation menus to guide yourself through the desired pages. Navigation menus are located in the header, footer, and dispersed linearly throughout the website beginning on the home page. To return back to the home page, simply click on the title in the header section.
